===================
 homebrew-yamlcast
===================

`Yamlcast <https://github.com/unnonouno/yamlcast>`_ is a C++ library handling YAML files.
This tap helps to install yamlcast onto OSX >= 10.9.

How to use
==========

.. code-block:: bash

   % brew tap pman0214/yamlcast
   % brew install yamlcast
